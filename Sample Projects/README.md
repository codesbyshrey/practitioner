# Sample Projects

Collection of small application builds, games, calculators, Python GUI projects, and frontend practice projects.

## Contents

- `2048`: Browser-based 2048 game project.
- `articleFocuser_tutorial`: Article focus tutorial project.
- `badquiz`: Quiz project variant kept as a practice or intentionally flawed implementation.
- `Brickbreaker`: Brick breaker game project.
- `Calculator`: Merged calculator practice project containing valid and invalid calculator variants.
- `Chess`: Chess project with local assets and chessboard.js dependency source.
- `ConverterPractice`: Converter practice project.
- `guitkintermodulepractice`: Tkinter GUI module practice.
- `hangmanBad`: Hangman project variant kept for review.
- `javascript`: JavaScript sample project collection.
- `LMS`: Library management system project, now merged with its work-in-progress ORV2 variants.
- `misc`: Miscellaneous sample projects and sandbox experiments.
- `Opik`: Merged Opik experiment project containing multimodal and text/vision optimization variants.
- `patientinfosys_workinprogress`: Patient information system work-in-progress project with ORV2 variant files merged under a subfolder.
- `reactwpy`: React/Python hybrid practice project with a Python app entrypoint.
- `rockpaperscissors`: Rock-paper-scissors project.
- plus 10 more nested folders.
