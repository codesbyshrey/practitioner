# Practitioner

Practice and experiment repository for small apps, algorithms, agent experiments, OpenAI API work, security notes, and coursework.

## Contents

- `Cursor`: Agent and IDE experiments spanning Python, Go, Java, and JavaScript implementations.
- `ds-algos`: Data structures and algorithms practice across JavaScript, Python, and PyCharm-era contest work.
- `JS-Security`: JavaScript security practice and notes.
- `northeastern`: Coursework, workshop files, and research-related material.
- `openai`: OpenAI API experiment folder; dependencies were removed and can be reinstalled from package metadata.
- `Sample Projects`: Collection of small application builds, games, calculators, Python GUI projects, and frontend practice projects.

## Revisit Notes

- `Cursor/index.html`, `Cursor/script.js`, and `Cursor/styles.css` form a memory-card game generated during Cursor/agent experimentation.
- Generated dependency/cache folders were removed and should be rebuilt locally when needed.
